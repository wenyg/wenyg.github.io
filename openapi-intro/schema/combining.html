组合

组合

json schema 有一些关键字用于将各个 schema 组合起来。

allOf

给定的数据必须满足所有的子模式

allOf:
    - type: string
    - maxLength: 5

匹配 

  • "short"

不匹配

  • "too long"

anyOf

给对的数据至少满足一个子模式

anyOf:
    - type: string
    - type: number

匹配

  • "Yes"

  • 42

不匹配

  • { "Not a": "string or number" }

oneOf

给定的数据必须满足其中的一个模式，而且也只能满足一个

oneOf:
    - type: number
      multipleOf: 5
    - type: number
      multipleOf: 3

匹配

  • 10

  • 5

不匹配

  • 2

  • 15

not

not:
    type: string

匹配

  • 42

  • { "key": "value" }

不匹配

  • "I am a string"
